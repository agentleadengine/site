Uses

The stack I actually use every day. Hardware, software, tools I'd pay for again.

Hardware

  • MacBook. Apple Silicon. Everything I do runs locally.
  • iPhone, mostly for capturing ideas and monitoring long-running tasks.

Editor + shell

  • Claude Code, my primary agent harness. The site you're reading was largely generated here.
  • zsh + Bun, standard macOS shell; Bun for anything Node-adjacent because it's 10× faster.
  • VS Code / Cursor, when I need a full IDE.

AI tools I pay for

  • Claude (claude.ai + API), daily driver
  • ChatGPT (OpenAI Plus), for comparisons and image gen
  • Perplexity, for fast research
  • Cursor, code editing with AI in the loop

Writing + notes

  • Notion, the long-term brain. Docs, projects, research notes.
  • Obsidian, for pure markdown notes I want local.
  • iA Writer, long-form writing.

Automation

  • n8n / Make / Zapier, pick based on the integrations needed. Lately leaning toward writing my own Claude Code skills instead.
  • GoHighLevel, opinionated, but it bundles what you'd otherwise glue together from five tools.

Adjacent infrastructure

  • Stripe, payments
  • Twilio. SMS/voice
  • Postgres, default database
  • Cloudflare. DNS + Pages when I need to ship a site fast
