XML sitemaps

📖 3 min readUpdated 2026-04-18

An XML sitemap is a file that lists the URLs you want search engines to know about. It's a hint, not a command, but a well-built sitemap speeds up discovery and helps with large sites.

What a sitemap contains

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://example.com/page1</loc>
    <lastmod>2026-04-18</lastmod>
    <priority>0.8</priority>
  </url>
  ...
</urlset>

Required vs optional fields

  • <loc>, required; absolute URL
  • <lastmod>, optional but useful; tells Google which pages changed
  • <changefreq>, essentially ignored by Google; don't bother
  • <priority>, also ignored; don't waste time setting

What to include

  • All canonical URLs that should be indexed
  • Static pages (/, /about, /services, etc.)
  • Blog posts, products, category pages

What NOT to include

  • Noindexed pages
  • Canonicalized duplicate URLs
  • Parameterized / filtered URLs
  • Redirect URLs
  • 404 pages
  • Pagination that's been handled with rel="next"/"prev"

Including garbage URLs sends mixed signals and wastes crawl budget.

Size limits

  • Max 50,000 URLs per sitemap
  • Max 50 MB uncompressed

Over these limits, split into multiple sitemaps + use a sitemap index file.

Sitemap index files

For large sites, create a sitemap index that lists child sitemaps:

<?xml version="1.0"?>
<sitemapindex xmlns="...">
  <sitemap><loc>https://example.com/sitemap-posts.xml</loc></sitemap>
  <sitemap><loc>https://example.com/sitemap-products.xml</loc></sitemap>
</sitemapindex>

Sitemap variants

  • Main sitemap. URLs
  • News sitemap, for Google News-approved sites
  • Image sitemap, inline image references inside main sitemap, or standalone for image-heavy sites
  • Video sitemap, for video content

How to submit

  1. Host at https://yourdomain.com/sitemap.xml (standard location)
  2. Reference in robots.txt: Sitemap: https://yourdomain.com/sitemap.xml
  3. Submit in Google Search Console → Sitemaps → add new
  4. Do the same in Bing Webmaster Tools

Dynamic sitemaps

On CMS-driven sites (WordPress, Shopify, etc.), the sitemap should update automatically when content changes. Most modern CMSes handle this via plugins (Yoast, RankMath) or built-in features.

Debugging sitemaps

  • GSC → Sitemaps report shows errors (invalid URLs, blocked pages, etc.)
  • Watch for "couldn't fetch" errors, sitemap URL itself is failing
  • Check "Discovered URLs" count vs. "Indexed URLs", large gaps mean indexing problems, not sitemap problems

When sitemaps don't help much

Small sites (<100 pages) with good internal linking don't strictly need a sitemap. Google will find everything anyway. But there's no cost to including one, so always do.
